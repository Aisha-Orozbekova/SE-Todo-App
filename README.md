# SE-Todo-App
Modern Task Manager built with Python and CustomTkinter with JSON data persistence. Premium To-Do App | Python SE Project | AIU College

# 📝 Premium Task Manager SE
Проект по дисциплине Software Engineering | Группа SCA-25C

## 🌟 Описание проекта
Это современное Desktop-приложение для управления списком дел. Основной упор сделан на качественный пользовательский интерфейс (UI) и удобство использования.

## ✨ Основные функции
* **Современный дизайн:** Интерфейс в темных тонах с использованием библиотеки `customtkinter`.
* **Умный учет:** Приложение автоматически считает общее количество задач и количество выполненных дел.
* **Надежное хранение:** Данные не пропадают после закрытия программы благодаря интеграции с `tasks.json`.

## 🛠 Технологический стек
* **Язык:** Python 3.12+
* **Интерфейс:** CustomTkinter (библиотека для создания современного UI)
* **База данных:** JSON (легковесный формат хранения данных)

## 🚀 Запуск проекта
1. Установите библиотеку: `pip install customtkinter`
2. Запустите основной файл: `python main.py`
