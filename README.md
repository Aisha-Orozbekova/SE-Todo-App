# 📝 Premium Task Manager SE
Современное приложение для управления списком дел, созданное в рамках проекта по Software Engineering.

## ✨ Особенности
* **Эстетичный Dark UI:** Интерфейс в стиле "Premium" с использованием библиотеки `customtkinter`.
* **Умные счетчики:** Приложение в реальном времени считает общее количество задач и сколько из них уже выполнено.
* **Надежность:** Все ваши задачи автоматически сохраняются в файл `tasks.json` и не пропадают после закрытия.

## 🚀 Технологии
* **Язык:** Python 3.12+
* **Библиотеки:** CustomTkinter, JSON

## 🛠 Как запустить
1. Установите библиотеку: 
   `pip install customtkinter`
2. Запустите код:
   `python main.py`
